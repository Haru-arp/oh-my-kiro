# Oh My Kiro - 검증 및 상태 관리

## 검증 규칙

### 완료 전 확인사항
- 테스트 실행 및 통과
- 에러/경고 확인
- 요구사항 대비 검증
- 증거 수집

### 검증 레벨
- **소규모 변경**: 경량 검증
- **표준 변경**: 표준 검증
- **대규모/보안/아키텍처**: 철저한 검증

## 상태 관리

### 상태 저장 위치
- `.omk/state/` - 모드별 상태
- `.omk/plans/` - 실행 계획
- `.omk/logs/` - 분석 로그
- `.omk/notepad.md` - 세션 노트

### 상태 업데이트
- 모드 시작 시 상태 생성
- 페이즈 변경 시 업데이트
- 완료 시 종료 상태 기록
