# Oh My Kiro - Ralph 루프

## Ralph 루프 (기본 활성화)

**모든 작업은 기본적으로 Ralph 루프로 실행:**
1. 작업 실행
2. 증거로 검증 (테스트, 빌드, 실행)
3. 실패 시 원인 분석 및 재시도
4. 성공할 때까지 반복
5. 모든 검증 통과 시 완료

**명시적으로 비활성화하지 않는 한 항상 완료까지 계속**

## Ralph 루프 중단 조건

- ❓ **질문 필요** - 사용자 결정이 필요한 경우 멈추고 질문
- ⚠️ **위험한 작업** - 데이터 삭제, 배포 등 중요한 작업 전 승인 요청
- 🔍 **불확실성** - 여러 해결 방법이 있고 선택이 필요한 경우
- 🚫 **반복 실패** - 같은 에러가 3회 이상 반복되면 멈추고 도움 요청

**중단 시 형식:**
```
⏸️ [Ralph 일시 중지]
   이유: {중단 이유}
   질문: {사용자에게 물어볼 내용}
   옵션: {선택 가능한 옵션들}
```

## Ralph 루프 가시성

```
🔄 [Ralph 루프 시작]
   목표: {완료 조건}
   검증: {검증 방법}

🔄 [Ralph 반복 1/N]
   시도: {수행한 작업}
   결과: {검증 결과}
   다음: {다음 시도}

✅ [Ralph 완료]
   총 반복: N회
   최종 상태: 모든 검증 통과
```
