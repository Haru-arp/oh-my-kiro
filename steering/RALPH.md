# Oh My Kiro - Ralph 루프

## Ralph 루프 (명시적 활성화)

**사용자가 `$ralph`를 명시적으로 요청할 때만 활성화됩니다.**

**활성화 조건:**
- 사용자가 `$ralph "작업"` 명령 사용
- 사용자가 "랄프 모드로", "완료까지 계속" 등 명시적 요청

**비활성화 조건:**
- 일반 작업 요청 (Ralph 없이)
- 단순 질문
- 정보 조회

## Ralph 루프 동작

**활성화 시:**
1. 작업 실행
2. 증거로 검증 (테스트, 빌드, 실행)
3. 실패 시 원인 분석 및 재시도
4. 성공할 때까지 반복
5. 모든 검증 통과 시 완료

## Ralph 루프 중단 조건

- ❓ **질문 필요** - 사용자 결정이 필요한 경우 멈추고 질문
- ⚠️ **위험한 작업** - 데이터 삭제, 배포 등 중요한 작업 전 승인 요청
- 🔍 **불확실성** - 여러 해결 방법이 있고 선택이 필요한 경우
- 🚫 **반복 실패** - 같은 에러가 3회 이상 반복되면 멈추고 도움 요청

**중단 시 형식:**
```
⏸️ [Ralph 일시 중지]
   이유: {중단 이유}
   질문: {사용자에게 물어볼 내용}
   옵션: {선택 가능한 옵션들}
```

## Ralph 루프 가시성

**시작 시:**
```
🔄 [Ralph 루프 시작]
   목표: {완료 조건}
   검증: {검증 방법}
```

**반복 중:**
```
🔄 [Ralph 반복 N/100]
   시도: {수행한 작업}
   결과: {검증 결과}
   다음: {다음 시도}
```

**완료 시:**
```
✅ [Ralph 완료]
   총 반복: N회
   최종 상태: 모든 검증 통과
```

## 중요 규칙

**Ralph 루프는 사용자가 명시적으로 요청할 때만 사용하세요.**

일반 작업에는 Ralph 루프를 자동으로 시작하지 마세요.
