# Oh My Kiro - 워크플로우 스킬

## 사용 가능한 스킬

- `$team` - 멀티 에이전트 팀 조정
- `$autopilot` - 자율 파이프라인
- `$ralph` - 지속 실행 루프
- `$ultrawork` - 병렬 작업 실행
- `$plan` - 구조화된 계획
- `$analyze` - 심층 분석
- `$tdd` - 테스트 주도 개발
- `$ecomode` - 토큰 효율 모드
- `$cancel` - 모드 취소
- `$ralplan` - 합의 계획
- `$visual-verdict` - 시각적 검증

## 팀 모드 페이즈

1. **team-plan**: 작업 분해
2. **team-prd**: 요구사항 정의
3. **team-exec**: 구현
4. **team-verify**: 검증
5. **team-fix**: 수정 (exec로 루프)
