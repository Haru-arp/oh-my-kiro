# 울트라워크 병렬 실행

여러 서브에이전트를 사용하여 독립적인 작업을 병렬로 실행

## 사용법

```
$ultrawork "모듈 A, B, C의 버그 수정"
$ultrawork "기능 1, 2, 3을 병렬로 구현"
```

## 워크플로우

1. 작업을 독립적인 하위 작업으로 분해
2. 병렬 서브에이전트 생성 (최대 6개)
3. 모두 동시 실행
4. 결과 병합
5. 결합된 출력 검증

## 규칙

- **모든 응답은 한국어로 작성**
- 작업이 실제로 독립적인지 확인
- 최대 6개 워커 제한
- 병합 전 각 결과 검증
