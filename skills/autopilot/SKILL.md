# 자동 조종

처음부터 끝까지 자율적인 파이프라인 실행

## 사용법

```
$autopilot "인증이 포함된 REST API 구축"
$autopilot "차트가 있는 대시보드 생성"
```

## 워크플로우

1. 요청 분석 및 페이즈로 분해
2. 실행: plan → implement → test → verify
3. 에러 자동 수정 및 반복
4. 모든 테스트 통과 시 완료

## 규칙

- **모든 응답은 한국어로 작성**
- 자동으로 에러 감지 및 수정
- 검증 없이 완료 주장 금지
