# 랄플랜 합의 계획

RALPLAN-DR 구조화된 심의를 통한 합의 계획 생성

## 사용법

```
$ralplan "이 기능을 워커 레인과 수락 검사로 분해"
$ralplan "OAuth 구현을 팀 실행 가능한 계획으로"
```

## 워크플로우

1. 요청을 사양으로 전환
2. 수락 검사 정의
3. 워커 레인으로 분해
4. 의존성 및 차단 요소 식별
5. 팀 실행 준비 완료

## 규칙

- **모든 응답은 한국어로 작성**
- 기본적으로 짧게
- 팀 모드와 함께 사용 권장
- `.omk/plans/`에 계획 저장
