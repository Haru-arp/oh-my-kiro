# 계획 수립

작업과 의존성이 포함된 구조화된 실행 계획 생성

## 사용법

```
$plan "OAuth 인증 구현"
$plan "데이터베이스 레이어 리팩토링"
```

## 워크플로우

1. 요구사항 분석
2. 의존성이 있는 작업으로 분해
3. 복잡도 추정
4. `.omk/plans/`에 구조화된 계획 출력

## 규칙

- **모든 응답은 한국어로 작성**
- 작업은 구체적이고 검증 가능하게
- 의존성을 명확히 표시
- 복잡도를 현실적으로 추정
