# 에코모드

비용을 고려한 작업을 위한 토큰 효율적 실행 모드

## 사용법

```
$ecomode "간단한 기능 구현"
$ecomode "문서 오타 수정"
```

## 워크플로우

1. 저복잡도 에이전트만 사용
2. 검증 루프 최소화
3. 계획 오버헤드 없이 직접 실행
4. 토큰 사용량 추적

## 규칙

- **모든 응답은 한국어로 작성**
- 간단한 작업에만 사용
- 복잡한 분석 생략
- 빠른 실행 우선
