# 랄프 지속 실행

작업이 완전히 완료될 때까지 계속하는 지속 실행 루프

## 사용법

```
$ralph "모든 TypeScript 에러 수정"
$ralph "완전한 테스트 커버리지 구현"
```

## 워크플로우

1. 작업 실행
2. 증거로 완료 검증
3. 미완료 또는 에러 발견 시 계속
4. 검증 완료까지 반복
5. 모든 기준 충족 시에만 중지

## 규칙

- **모든 응답은 한국어로 작성**
- 검증 없이 완료 주장 금지
- 최대 반복 횟수 준수
- 각 반복마다 진행 상황 보고
