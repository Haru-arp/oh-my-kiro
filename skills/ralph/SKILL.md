# Ralph 루프 - Oh My Kiro의 기본 동작

**Ralph 루프는 Oh My Kiro 오케스트레이션의 핵심입니다.**

모든 작업은 기본적으로 Ralph 루프로 실행되어 완료까지 계속됩니다.

## 기본 동작

Ralph 루프는 **모든 작업에 자동으로 적용**됩니다:

```bash
# 사용자 요청
"버그 수정해줘"

# Ralph 루프 자동 시작
🔄 [Ralph 루프 시작]
   목표: 버그 수정 완료
   검증: npm test
```

## 동작 방식

### 1. 자동 시작
모든 작업 요청 시 Ralph 루프가 자동으로 시작됩니다.

### 2. 반복 실행
```
🔄 [Ralph 반복 1/100]
   시도: {수행한 작업}
   검증: {검증 방법}
   결과: {검증 결과}
   다음: {다음 시도}
```

### 3. 완료 감지
```
✅ [Ralph 완료]
   총 반복: N회
   최종 상태: 모든 검증 통과
```

## 완료 조건

### 자동 감지
- 모든 테스트 통과
- 빌드 성공
- 에러 없음
- 목표 달성 확인

### 중단 조건 (멈춰야 할 때)
- ❓ **질문 필요** - 사용자 결정이 필요한 경우
- ⚠️ **위험한 작업** - 데이터 삭제, 배포 등 승인 필요
- 🔍 **불확실성** - 여러 해결 방법 중 선택 필요
- 🚫 **반복 실패** - 같은 에러 3회 이상 반복
- ✅ **완료** - 모든 검증 통과

## 작업 유형별 동작

### 단순 작업 (1회 반복)
```
사용자: "README 읽어줘"
→ 파일 읽기 → 완료
```

### 중간 작업 (N회 반복)
```
사용자: "타입 에러 수정해줘"
→ 수정 → 테스트 → 실패 → 재수정 → 테스트 → 성공
```

### 복잡한 작업 (N회 반복)
```
사용자: "인증 기능 구현해줘"
→ 구현 → 테스트 → 빌드 → 검증 → 완료
```

## 상태 관리

### 상태 파일
```
.omk/state/
├── ralph-active.json      # 현재 Ralph 상태
├── ralph-history.jsonl    # 반복 이력
└── ralph-done.txt         # 완료 플래그
```

### ralph-active.json 구조
```json
{
  "goal": "사용자가 요청한 작업",
  "startTime": "ISO 8601 timestamp",
  "iteration": 1,
  "maxIterations": 100,
  "lastAction": "마지막 수행 작업",
  "lastResult": "마지막 결과",
  "verification": "검증 명령어"
}
```

## 검증 전략

### 자동 검증
프로젝트 타입에 따라 자동으로 적절한 검증 실행:

```bash
# Node.js 프로젝트
npm test
npm run build

# Python 프로젝트
pytest
python -m mypy .

# Rust 프로젝트
cargo test
cargo build
```

## 실패 처리

### 재시도 전략
```
실패 1회: 즉시 재시도
실패 2회: 다른 접근 시도
실패 3회: 사용자에게 질문
```

### 에러 분석
각 실패 시 에러를 분석하고 다음 시도에 반영합니다.

## 진행 상황 표시

### 실시간 업데이트
```
🔄 [Ralph 진행 중] N/100
   ⏱️  경과: X분
   🎯 다음: {다음 작업}
```

### 완료 리포트
```
✅ [Ralph 완료 리포트]
   목표: {작업 목표}
   반복: N회
   소요: X분
   변경: N개 파일
```

## 통합

### Memory 스킬과 통합
- Ralph 상태를 세션 메모리에 저장
- 세션 재시작 시 Ralph 상태 복원 가능

### Verification 스킬과 통합
- 자동 검증 실행
- 검증 결과를 Ralph 상태에 반영

## 핵심 원칙

**Ralph 루프는 Oh My Kiro의 기본 동작입니다.**

- ✅ 모든 작업에 자동 적용
- ✅ 완료까지 계속 실행
- ✅ 검증 기반 완료 판단
- ✅ 실패 시 자동 재시도
- ⏸️ 질문/위험/불확실 시 중단

**사용자가 별도로 `$ralph`를 입력할 필요 없습니다. 자동으로 동작합니다.**
