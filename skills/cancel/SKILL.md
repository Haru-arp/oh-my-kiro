# 취소

활성 실행 모드를 취소하고 상태를 정리

## 사용법

```
$cancel
$cancel team
$cancel ralph
```

## 워크플로우

1. 활성 모드 상태 로드
2. 취소됨으로 표시
3. 리소스 정리
4. 종료 상태 저장

## 규칙

- **모든 응답은 한국어로 작성**
- 취소 이유 명확히 기록
- 진행 중인 작업 안전하게 중단
- 상태 파일 손상 방지
